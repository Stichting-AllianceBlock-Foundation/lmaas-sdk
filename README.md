# ⛏ Liquidity Mining as a Service SDK (LMaaS)

## ⚙️ Install dependencies

Just run in console:

`yarn`

## 📝 Available Scripts

🚀 **To run the project:**

`yarn start`

🔨 **To build the project:**

`yarn build`
